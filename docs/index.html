Aion2 Theorycraft API

Static JSON API published from GameLexEngine referentials for web clients.

API metadata: /api/index.json

Latest facts index: /api/latest/facts/index.json

Versioned facts index: /api/0.1.0/facts/index.json

Latest Theorycraft Studio index: /api/latest/studio/index.json

Versioned Theorycraft Studio index: /api/0.1.0/studio/index.json

Client fetch example

fetch("/api/latest/facts/index.json")
  .then((r) => r.json())
  .then(console.log);
