# Aion 2 Theorycraft

Welcome to the official public distribution repository for Aion 2 Theorycraft. This repository serves as the public delivery layer of the Theorycraft-Labs ecosystem, providing tools, data, and resources for the Aion 2 community.

## About

This repository contains theorycraft tools and data for Aion 2, including:
- Character build calculators
- Stat optimization tools
- Damage calculation models
- Game data and resources

## Installation

### Prerequisites

To use the tools in this repository, you may need:
- A modern web browser (for web-based tools)
- Python 3.8+ (for Python-based tools)
- Node.js 16+ (for JavaScript-based tools)

### Quick Start

```bash
# Clone the repository
git clone https://github.com/Theorycraft-Labs/aion2-theorycraft.git
cd aion2-theorycraft

# Follow specific installation instructions in the documentation
```
