# Aion2 Theorycraft

Static web project focused on Aion 2 theorycraft.

## What is published

- Studio web client (GitHub Pages root)
- JSON API under `docs/api/` generated from GameLexEngine

## Data origin

API data is extracted from publicly available online sources and transformed into structured referentials for analysis and navigation use.
